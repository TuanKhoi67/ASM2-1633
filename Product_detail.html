LEGO Mercedes-AMG F1 W14 E Performance

$999.00

Build and display this LEGO Mercedes-AMG F1 W14 E Performance model. Features intricate details and includes minifigures.

Description:

Features include steering that can be controlled using the steering wheel or with the knob on top of the car. Remove the engine cover to reveal the differential and the 6-cylinder engine with moving pistons. With amazing attention to detail, the model also features an opening wing inspired by the real-world car's DRS.
